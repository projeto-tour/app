Documentação de Software do Projeto Partiu!

O objetivo deste documento é dispor de forma organizada todos os diagramas, modelos, código fonte, tecnologias, requisitos, etc.. que foram utilizados para o desenvolvimento do software Partiu!
  • 01-introducao Este documento representa a especificação de requisitos para a construção de um aplicativo para a plataforma mobile. Sua função é apoiar a gestão de viagens, hospedagens, deslocamentos, agendamentos, passeios, excursões, entre outros eventos turísticos ou de negócio com o intento de promover o turismo, divulgando e facilitando sua exploração.
  • 02-resumo Este documento visa esclarecer o problema que o software resolve dentre a gestão de turismo assim como abordagem das funcionalidades que é oferecido ao usuário.
  • 03-descricao-detalhada Este Documento tem uma descrição detalhada do software e suas funcionalidades.
  • 04-tecnologias-utilizadas O documento preve das tecnologias que foram utilizadas para desenvolvimento do software e uma descrição de cada uma delas.
  • 05-documentacao do sistema Este Documento tem uma descrição detalhada do software e suas funcionalidades, recursos computacionais necessários para utilização, funcionalidades, plataformas utilizadas e compatíveis de execução do software.
  • 06-tutorial Neste capítulo temos o tutorial de uso do software, deixando explicito o funcionamento de todas telas com suas respectivas funcionalidades, apresentando também as restrições existentes para acesso a cada interação, não deixando assim o usuário obter qualquer utilização indevida ou errada.
